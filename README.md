# 易经六十四卦 · 每日一卦

一个简洁美观的易经六十四卦小程序，支持每日起卦，随时查阅。

## 功能特点

- ✅ **每日一卦** - 基于日期种子，同一天得到同一卦
- ✅ **重新起卦** - 点击按钮随机抽取
- ✅ **完整六十四卦** - 包含全部卦象、卦辞、爻辞、解释
- ✅ **可视化卦象** - 正确绘制阴阳爻（从下往上排列）
- ✅ **点击查看详情** - 弹出查看完整六爻
- ✅ **响应式设计** - 手机/电脑都能流畅浏览
- ✅ **纯静态** - 零后端，直接部署

## 部署

### Vercel (推荐)

1. Fork 这个仓库
2. 在 Vercel 导入 `idcim/hexagrams`
3. 点击 Deploy，搞定！

### 本地运行

```bash
# 克隆项目
git clone https://github.com/idcim/hexagrams.git
cd hexagrams

# 使用 Python 简单HTTP服务器
python3 -m http.server 3000
# 访问 http://localhost:3000
```

### 任何静态文件托管

直接把这四个文件上传就行：
- `index.html`
- `style.css`
- `hexagrams.js`
- `app.js`

## 使用方法

1. 打开网页，顶部自动显示**今日卦象**
2. 点击「查看详情」看完整卦辞和六爻
3. 点击「重新起卦」随机抽一卦
4. 下方列表点击任意卦象查看详情

## 数据来源

数据来源于《周易》易经，保留原文，并添加了白话解释方便理解。
